# Homework-9
Node.js and ES6+

Objective: Command-line application that dynamically generates a PDF profile from a GitHub username.

# Developer Profile Generator
The user will be prompted for a favorite color, which will be used as the background color for cards.

![homework-9-nodejs](https://user-images.githubusercontent.com/44029053/73587439-be2c3d00-4489-11ea-8def-21a89634413a.gif)

The PDF will be populated with the following:

* Profile image
* User name
* Links to the following:
* User location via Google Maps
* User GitHub profile
* User blog
* User bio
* Number of public repositories
* Number of followers
* Number of GitHub stars
* Number of users following

# User Story
```
AS A product manager

I WANT a developer profile generator

SO THAT I can easily prepare reports for stakeholders
```
Built With: Javascript, Node.js, npm modules
